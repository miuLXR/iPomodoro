## 一、产品概述

### 1.1 产品定位
FocusTomato 是一款基于番茄工作法的专注计时工具，帮助用户提升工作效率，培养专注习惯。

### 1.2 核心价值
- 简单易用的番茄计时器
- 可视化的专注数据统计
- 无干扰的纯净体验
- 完全免费，无广告

### 1.3 目标用户
- 学生：需要专注学习
- 职场人士：提高工作效率
- 自由职业者：管理工作时间
- 任何希望提升专注力的人群

## 二、功能需求

| 功能点 | 描述 | 优先级 |

```

                                    ↓
                            计时结束 → 通知+音效
                                    ↓
                            自动切换到休息模式
```

## 四、数据模型

### 4.1 核心数据结构

```swift
// 番茄记录
struct TomatoRecord: Codable {
    let id: UUID
    let date: Date
    let duration: Int  // 专注时长(分钟)
    let taskName: String?
    let isCompleted: Bool
}

// 用户设置
struct UserSettings: Codable {
    var focusDuration: Int = 25      // 专注时长(分钟)
    var shortBreakDuration: Int = 5   // 短休息时长
    var longBreakDuration: Int = 15   // 长休息时长
    var soundEnabled: Bool = true     // 音效开关
    var vibrationEnabled: Bool = true // 振动开关
    var autoStartBreak: Bool = true   // 自动开始休息
    var longBreakInterval: Int = 4    // 几个番茄后长休息
}

// 计时器状态
enum TimerState {
    case idle, running, paused
}

enum SessionType {
    case focus, shortBreak, longBreak
}
```

### 4.2 数据存储

- 使用 `UserDefaults` 存储用户设置
- 使用 `SwiftData` 或 `CoreData` 存储番茄记录
